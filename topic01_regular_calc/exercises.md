## ④デバッグ演習

第1問（境界値バグ）「1〜N の合計」のつもり。何が起きる？

    goukei = 0
    for i in range(1, N):
        goukei += i
    print(goukei)

第2問（無限ループ）素数判定のつもり。実行するとどうなる？

    n = 7; i = 2; is_prime = True
    while i < n:
        if n % i == 0:
            is_prime = False
    print(is_prime)
