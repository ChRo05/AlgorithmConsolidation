## ③トレース演習（添字1始まり）

第1問（基本：行ごとの合計） A（2行3列）＝
行1：[2, 0, 5] ／ 行2：[1, 3, 4]

各行の合計 gyokei を求める（行が変わるたびに gyokei ← 0 に戻す）。

| i | j | A[i][j] | gyokei |
|---|---|---------|--------|
| 1 | 1 |         |        |
| 1 | 2 |         |        |
| 1 | 3 |         |        |
| 2 | 1 |         |        |
| 2 | 2 |         |        |
| 2 | 3 |         |        |

第2問（境界値：負数・0入りの列ごとの合計） A（3行2列）＝
行1：[3, -1] ／ 行2：[0, 4] ／ 行3：[-2, 5]

各列の合計 retukei を求める（外側が列 j、内側が行 i になることに注意）。

| j | i | A[i][j] | retukei |
|---|---|---------|---------|
| 1 | 1 |         |         |
| 1 | 2 |         |         |
| 1 | 3 |         |         |
| 2 | 1 |         |         |
| 2 | 2 |         |         |
| 2 | 3 |         |         |

🗣️ OUTPUT：トレース後、表を見ずに「行ごとの合計と列ごとの合計で、二重ループの何が入れ替わるか」を2文で説明せよ。

## ④デバッグ演習

第1問（長さの取り違え）2行3列の合計のつもり。エラーは出ないのに答えが合わない。なぜ？

    A = [[2, 0, 5], [1, 3, 4]]
    goukei = 0
    for i in range(len(A)):
        for j in range(len(A)):
            goukei += A[i][j]
    print(goukei)

第2問（初期化の位置バグ）行ごとの合計のつもり。2行目の答えがおかしい。なぜ？

    A = [[2, 0, 5], [1, 3, 4]]
    gyokei = 0
    for i in range(len(A)):
        for j in range(len(A[i])):
            gyokei += A[i][j]
        print(i + 1, "行目の合計:", gyokei)

🗣️ OUTPUT：各バグを「生徒がやりがちなミス」として、なぜ起きるかを一言で言語化せよ。特に第1問が“エラーが出ないぶん たちが悪い”理由に触れること。
