# トピック3：解答・解説（＋説明スクリプト）

## ③トレース解答

第1問 S = "LEVEL"（n=5） → 回文（True）

| i | S[i] | S[n−i+1] | 一致? | kaibun |
|---|------|----------|-------|--------|
| 1 | L | L (S[5]) | ✓ | True |
| 2 | E | E (S[4]) | ✓ | True |

中央 S[3]=V は比較せず（奇数長の中央は相手が自分自身なので必ず一致）。

第2問 S = "ABCA"（n=4） → 非回文（False）

| i | S[i] | S[n−i+1] | 一致? | kaibun |
|---|------|----------|-------|--------|
| 1 | A | A (S[4]) | ✓ | True |
| 2 | B | C (S[3]) | ✗ | False |

要点：偶数長は全ペア（n÷2 組）を比較。i=2 で不一致が出た時点で False 確定、以降 True に戻らない。

🗣️ 説明スクリプト：「文字列を鏡と思う。左端と右端を1組ずつ突き合わせ、1組でも違えば回文でない。中央まで n÷2 回見れば十分」

## ④デバッグ解答

第1問 原因：s[n - i] は添字が1つズレ（正しくは s[n - 1 - i]）。i=0 で s[n]＝s[5] は存在せず IndexError。範囲も range(n // 2) で十分。
修正：

    for i in range(n // 2):
        if s[i] != s[n - 1 - i]:
            kaibun = False

第2問 原因：不一致のとき match = False とすべきを match = True にしている。だから常に True。
修正：

    if text[i + j] != pat[j]:
        match = False
